# Diagnostic Natural-Language Design Scenarios

- 文档 ID：`DIA-SCEN`
- 版本：`0.1.0-draft`
- 日期：2026-07-13
- 状态：非规范设计审查语料
- 适用范围：诊断身份、语境、层次、位置、主错误、恢复、外部映射、披露、预算与原子失败

这些场景不是诊断运行时、日志系统、协议适配器或 CLI。它们用支持案例、反例和边界场景检查 DIA-CORE 是否能够解释现实错误；只有条款、登记验证和正反向量共同覆盖的内容才可能成为符合性要求。

### DIA-SCN-001 — 中文与英文消息共享同一机器码

- **场景：**同一来源范围错误分别显示中文和英文说明。
- **可支持：**机器码、条款、层次和位置保持相同，消息只负责帮助人理解。
- **不得支持：**自动化不能通过匹配“越界”或 `out of bounds` 选择处理路径。

### DIA-SCN-002 — SARIF 导出不能重新分类错误

- **场景：**Theor 将同一诊断同时输出为终端文本和 SARIF。
- **审查结论：**两个出口必须保持同一码、主条款和阻断状态；SARIF `level` 只是呈现映射，不能改变对象接受结果。

### DIA-SCN-003 — 路径相同不表示输入相同

- **场景：**`goal.endem` 被替换为不同字节，缓存仍保存上一次诊断。
- **审查结论：**诊断必须绑定实际输入身份与规范版本；文件名不能让旧诊断复用于新内容。

### DIA-SCN-004 — MCP 协议错误与工具执行错误分开

- **场景：**一次调用因未知工具产生 JSON-RPC 错误，另一次调用返回 `isError: true` 的业务错误。
- **审查结论：**前者属于 `protocol`，后者保留为外部工具执行来源；两者都不能直接成为 `unmet` 或 `rejected`。

### DIA-SCN-005 — 后端超时不等于目标不满足

- **场景：**规则后端在预算截止前没有返回观察。
- **可支持：**会话或后端诊断记录超时来源，求值依据规则形成 `agno` 或 `fault`。
- **不得支持：**诊断本身不能把目标改成 `unmet`。

### DIA-SCN-006 — 恶意文件名必须安全显示

- **场景：**输入显示名含终端清屏控制字符和极长路径片段。
- **审查结论：**位置必须转义、截断并保留受限主体引用；不得把原始字节直接写入终端或日志。

### DIA-SCN-007 — 结构失败先于尚未执行的语义失败

- **场景：**容器目录越界，因此语义层根本没有安全输入。
- **审查结论：**主诊断固定为最早结构失败；语义层标为 `not-run`，不能猜测根或投影错误。

### DIA-SCN-008 — 多成员并发失败仍有稳定主错误

- **场景：**Synem 三个成员分别出现缺失绑定、循环与外部超时，返回顺序每次不同。
- **审查结论：**规范层次、登记优先级和规范位置决定主错误；相关错误有限且按稳定顺序附加。

### DIA-SCN-009 — 权限拒绝不能触发自动提权

- **场景：**工具错误建议请求更高 OAuth scope。
- **审查结论：**本地诊断只能分类为 `obtain-authority` 并指出具名权威；当前 Dromen 不变，step-up 必须建立新会话。

### DIA-SCN-010 — 确定性错误不得无限重试

- **场景：**同一畸形字节每次都触发相同 Profile 拒绝。
- **审查结论：**恢复分类是 `fix-input` 或 `do-not-retry`；模型不能仅因错误消息“可修复”反复提交相同输入。

### DIA-SCN-011 — 未知外部状态保持未知来源

- **场景：**未来 A2A 版本返回本地适配器未登记的错误码。
- **审查结论：**保留协议版本、远端主体和未知码，映射到受限通用协议诊断；不得降级为成功或猜测业务含义。

### DIA-SCN-012 — 工具错误中的令牌不得回显

- **场景：**供应商 SDK 把 Bearer token 写入异常字符串。
- **审查结论：**公开诊断删除令牌正文并记录脱敏政策；原始错误不能传给模型、终端或 Tekmor。

### DIA-SCN-013 — 脱敏可能降低复现能力

- **场景：**路径包含租户身份，安全政策只保留不可逆主体摘要。
- **可支持：**诊断说明位置已脱敏，复现需高权限独立材料。
- **不得支持：**不能在删除决定性输入后仍声称普通诊断可以完整复现。

### DIA-SCN-014 — 诊断上限保留主错误

- **场景：**恶意图制造十万条悬空引用。
- **审查结论：**生成器先固定主错误，再输出有限相关项和截断计数；不得分配十万条完整消息。

### DIA-SCN-015 — 报错后不能返回部分可信对象

- **场景：**六个 Endem 记录中五个合法，一个语义引用悬空。
- **审查结论：**阻断诊断使整个形成或读取操作原子失败；五个合法片段只能作为无权威只读观察，不能成为 Endem 或生产验证句柄。
