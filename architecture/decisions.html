架构决策

说明当前采用的 Endem 制品体系、单一命令面、组件边界和旧决定的取代状态。

ADR-0008 AcceptedSupersession TrackedABI Unfrozen

当前权威决定

ADR-0008：Endem 制品与单一命令面是当前对象、组件、命令、生命周期和首条实现切片的权威架构决策。它冻结名称和职责，不冻结二进制 ABI、文本语法或安装接口。

旧 ADR 继续作为设计历史存在，但被取代的术语、路由和工具数量不能反向约束当前实现。仍被保留的安全原则已经映射到 Core、Reader、Runner、Frame 和 Witness。

ADR-0008 冻结的内容

决定采用的边界直接影响
最小目标制品一个 Endem 只能有一个根 aim，并保留 say、must、done 与 open。来源格式、Core、规范向量和错误语义。
同一制品状态Endem 沿 open → bound → sealed 演进，不建立平行对象家族。身份、摘要、发布、签名和生命周期。
组合闭包Weave 只在多个 Endem 需要解析关系时出现；权限只能收紧。bind、依赖锁定、冲突和披露。
运行非文件视图Frame 只存在于会话内，不携带到磁盘或跨会话。Runner、能力句柄、上下文和隔离。
有限证据主张Witness 保存可定位依据，但不自动证明结论或形成 accepted。来源、绑定、签名、运行、验收与审计。
唯一 CLIendem 提供 form/check/bind/pack/seal/see/run/test。工具项目、手册、安装包和自动化接口。
独立与隔离see 不共享生产解析；run 不共享编译器权限域。仓库依赖方向、测试和安全边界。

排除的捷径

  • 不把任意自然语言或模型 JSON 直接视为 Endem。
  • 不为查看、符号、大小、归档、裁剪、跟踪和评估各建一个产品。
  • 不使用环境路径、当前目录或网络“最新版本”解析正式依赖。
  • 不让通用对象变换器任意改写 aim、must、done 或 open。
  • 不把签名、Witness 数量、模型评分或 Runtime 自述当作任务完成。
  • 不把 Frame 序列化为可复制的权限对象。

外部机制的采用边界

GNU、MCP、A2A 和智能体工程资料只用于核对可复用机制，不决定 Endem 语义或成熟度：

  • GNU readelf 与 BFD：https://www.sourceware.org/binutils/docs/binutils/readelf.html、https://sourceware.org/binutils/docs/bfd.html
  • MCP 2025-11-25：https://modelcontextprotocol.io/specification/2025-11-25
  • A2A 1.0.1：https://a2a-protocol.org/v1.0.1/specification/
  • OpenAI Harness Engineering：https://openai.com/index/harness-engineering/

变更规则

任何改变 Endem 五组语义、根 aim 唯一性、状态机、Weave 组合、Frame 非文件边界、Witness 有限主张或唯一命令面的提案，都必须新增 ADR，并同步规范、组件、生命周期、手册和测试向量。
