ADR-0019
先定义测量
再比较阈值

分数只有在构念、总体、程序、单位、聚合器和不确定度都明确时才可比较；基准排名、仪表盘显示值与模型置信度都不能直接成为目标结论。

当前策略测量口径先于分数不改变线格式尚未实现

用一次延迟检查读懂测量链

“发布窗口内成功请求的 p95 响应时间不高于 300 毫秒”不是一条仪表盘查询。它是从目标总体、观察程序、统计汇总到阈值判断的一条完整证据链。

构念与值角色从请求发出到成功响应完成的持续时间。“性能”“体验”或仪表盘指标名。
总体与窗口发布窗口内完成的生产成功请求。只挑选低延迟请求或移动时间窗口。

先定义测量什么和适用于谁

测量谓词必须同时固定构念、适用总体、可观察标准、预期用途和具名权威；分数不能反过来替这些条件作决定。

主张最低契约不能推出
测量构念关系、值角色、直接可观察标准、用途和具名语义权威。代理任务、模型裁判或指标名自行定义“质量”“安全”或“智能”。
固定测试集 population=fixed精确题目、样本、试验、评分程序和随机性来源。未测试题目、用户、领域或实际部署上的普遍能力。
推广总体 population=generalized目标总体、estimand、抽样假设、预注册统计模型和适用限制。通用智能、所有真实任务或未声明人群上的表现。
代理测量代理任务与目标构念的关系、偏差风险和适用范围。排行榜名次或裁判模型置信度直接成为目标事实。

术语边界：fixed 也用于 ADR-0016 的时间范围。代码、日志和口头说明必须写成 population=fixed 或 time_scope=fixed，中文先说“固定测试集”或“固定时刻范围”。estimand 首次出现时写成“目标统计量（estimand）”，不把难读的英文单独作为人类界面标签。

测量记录是关于事态的证据，不是事态本身。阈值比较形成满足结果，也不会自动产生最终接受或动作授权。

再固定程序、单位与聚合器

登记项必须固定典型失败
数值与单位有限数值域、量纲、规范单位、观察单位、换算权威、版本和误差。NaN、无穷、隐式百分比缩放或不兼容量纲。
测量程序生产者、方法版本、窗口、纳入排除、最小样本、缺失和删失处理。删除失败记录、移动窗口或样本不足仍给出结论。
point / count / sum / min / max观察身份、时间窗、数值域；累计量另需重置与重叠规则。把不同窗口或生产者的值直接相加。
arithmetic_mean / fraction均值权重；比例的分子、分母和零分母政策。把不同权重均值合并，或把百分数与比例混用。
quantile / model_estimate分位数 q、算法和最大误差；模型估计的 estimand、模型、假设与不确定度。更换插值或分桶，或把自报置信度当作统计区间。

非线性或分段换算还必须固定方向、定义域、值域和误差。换算只有在声明域内确定、量纲兼容且满足所需单调性时才能进入比较；舍入只能发生在比较完成后的显示阶段。

阈值必须连同不确定区间比较

采集、换算、统计或比较程序违反契约。fault单位或分位数算法不可定位。

比较器必须明确为 lt / le / gt / ge / eq，并绑定阈值、单位、原始观察、换算、聚合器、区间与实际 Iknem。把 0.76 先显示舍入为 0.8，再与 0.8 比较，必须拒绝。

AI 基准和遥测工具只提供什么

外部资料可采用的机制Noemion 不继承
NIST AI 800-2 Initial Public Draft按评估目标、基准选择、执行、分析和报告组织自动化模型与 Agent 评测。初始公开草案不是字段标准，也不提供项目合格阈值。
NIST AI 800-3区分固定基准准确率与泛化准确率，并显式选择 estimand、统计假设和不确定度。不把 GLMM 冻结为唯一算法，推广总体也不等于所有现实用途。
OpenTelemetry Metrics 数据模型保存单位、数据点类型、时间窗、temporality、分桶边界、重置和无记录值。同一事件可形成不同 View；遥测流本身不选择 krin、总体或阈值。
Prometheus 直方图与摘要指南展示分桶、分位数算法、时间窗和聚合位置怎样改变误差与可合并性。仪表盘查询不能事后替换已授权方法。
GNU Units 手册检查量纲、定义域、非线性换算和非唯一逆；units --check 可发现部分定义错误。单点逆函数检查不保证换算全域正确，也不能决定业务总体、聚合器或阈值。

当前还不能编码或执行什么

现行十二个测量提案向量覆盖六个允许分类和六个确定拒绝，只验证 END-MSR 四条抽象条款。本决定不增加 END-P1 字段，也不表示遥测采集器、基准运行器、统计引擎、Drasor 或求值器已经实现。

单位词典及版本、跨叶测量公式、多重比较、顺序检验、基线漂移、模型裁判校准、统计模型登记、隐私保护测量、物理字段和规范字节仍需新 Profile、威胁分析与实现证据。
