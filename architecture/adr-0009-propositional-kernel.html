ADR-0009
Endem 命题投影语义核

保存从需求分类转向命题结构的决策历史；五组名称与目标力量模型已由 ADR-0010 取代。

Superseded by ADR-00102026-07-12Historical SemanticsEncoding Unfrozen

直接结论

现行决定：ADR-0010：原生词汇与事态分层采用六个语义面，并分开中性事态、目标方向和结构化观察。以下五组旧词只用于解释迁移。

这份 ADR 当时把需求分类推进为命题投影：say / aim / must / done / open 改为 say / mean / case / when / open。ADR-0010 后来又用六个现行语义面取代了这两组名称。

核心不变量：一个 Endem 只有一个根 case。mean 与 case 共享稳定符号和关系位置；它们的图结构显示形式，不设置由制品自填的 logical_form、valid 或 true 字段。

五组投影语义

组回答的问题失败边界
say来源中出现了哪些可重定位记号？来源不可重放、摘要不匹配或推断伪装成原文。
mean这些记号怎样绑定符号、指称、关系、作用域和力量？模型摘要被当作意义、指称无来源或作用域被静默选择。
case哪一种可能事态应成为、保持或不得成为现实？没有根命题、出现多个独立根或只有实体清单没有关系。
when哪些观察支持满足、反驳、未知或错误，谁有权决定？命题自证、模型评分替代 Witness 或未知被当作满足。
open投影、指称、关系、测量或权威还有什么未确定？只保留最高概率候选、运行时默认或声称编码不可说之物。

从记号到可能事态

say · source signs
mean · projection
case · possible situation
when · fulfillment conditions
Witness · observation
Decision
  • say 保存表达事实，不宣称解释正确。
  • mean 保存已授权投影；无法选择的候选继续进入 open。
  • case 保存对象—关系图。Phase 1 至少区分 seek、keep 与 avoid。
  • when 把关系节点绑定到观察和决定规则，但不把观察或结论写成命题的自我真值声明。

三种不能混淆的未确定

状态含义处理
no-sense记号组合违反 Profile，或没有任何允许的投影。form 拒绝，不产生合法 Endem。
open存在可表达候选，但规则与授权不足以选择。保留候选、来源和解决权限；不能执行阻断性未决项。
unknown意义与满足条件已经确定，但外部观察不足。Runner 保留未知，不把它改成 unmet 或 met。

工程满足状态还区分 met、unmet 与 error。这些状态不是哲学真值；最终 accepted 还要求 Witness 完整、权威匹配并且没有阻断性 open。

《逻辑哲学论》的采用边界

原书问题工程采用明确拒绝
事实是对象的结合（1–2）case 使用对象—关系结构。不断言世界由不可再分逻辑原子组成。
图像通过元素关系接触实在（2.13–2.21）mean 记录符号、指称和投影。不假定自然语言与现实天然同构。
命题是有结构的事实（3.14–3.1432）关系位置和组合边进入规范内容。不把词袋或五段文本称为命题结构。
理解关联到真时的情况（4.022–4.024、4.431）case 与 when 分别保存可能事态和满足条件。不把开放目标缩为封闭二值真值表。
逻辑形式只能显示（2.172、4.12–4.1212）结构由 Core 和独立 Reader 分别验证。不设置自称有效或真实的载荷字段。
为可说者划界（4.112–4.116、7）分开 no-sense、open 和 unknown。不声称格式能够保存不可说之物。

本次对照路德维希·维特根斯坦著、贺绍甲译《逻辑哲学论》（商务印书馆，2009 年第 1 版）及 Wittgenstein Project 开放编号校本。

对 ADR-0008 的影响

在本 ADR 作出时，ADR-0008 仍决定 Endem、Weave、Frame、Witness、三个状态、单一 CLI、八个动作以及三个权限域。本 ADR 只取代其中的五字段分类、根 aim 和对应实现切片；ADR-0010 后来又取代其余历史名称。

开发阶段不提供旧字段别名或转换垫片。所有规范、手册、示例、视觉和测试必须使用新语义核；历史 ADR 只用于解释迁移。

仍未冻结的内容

这份历史记录没有冻结物理记录编号、文本语法、符号 ID、量化、时间、求值语言或稳定 ABI。当前未决范围以 ADR-0010 和开放问题页为准。
