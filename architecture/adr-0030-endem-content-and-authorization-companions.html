项目/架构设计/架构决策/ADR-0030

ADR-0030
内容身份不变
不等于授权仍然有效

让 Endem 文件稳定回答“对象是什么”，让外部伴随关系单独回答“谁在什么范围和截止点允许怎样使用它”；授权过期或撤销不能改写历史内容身份。

当前草案END-CORE 职责澄清内容身份稳定尚无伴随 Profile

用同一份 Endem 理解两种变化

假设一份 END-P1 形成制品在周一获得发布团队确认，周三该授权被撤销。文件字节没有变化，因此内容身份保持不变；周三的“是否仍可用于发布”必须重新评估。

授权过期、撤销或政策版本失效不变相对新截止点的未满足或未求值结论
新增签名、Iknem 或最终决定不变新的签名、证据或决定关系；各自保留结果域

未来从形成版派生裁剪发布版改变新 Profile、新身份、来源关系和重新验证结果

当前策略：END-P1 仍只保存六个语义记录。文件可以独立通过容器与 Profile 检查，但文件内权威名称不能证明授权已经发生。

为什么权威名称只能是待核对选择器

文件内表达它实际能够说明什么完整内容接受还缺什么
projection.kind候选来自哪类确定性规则或具名权威规则版本、声明输入域、输出位置与实际执行结果
projection.id指向一个待核对规则、角色或主体名称当前主体资格、精确候选、语义位置、范围和截止点
decision_authority指出谁应当作出判断该主体是否看过当前安全视图并形成有效决定
required_iknem指出内容要求的证据选择器实际 Iknem、有效性、覆盖度、撤销状态和适用判据
签名者或本地 ACL 名称为外部验证或本地政策提供查找线索签名陈述、信任政策、授权范围和依赖方决定

读取器不得通过当前环境、网络查询、默认目录或缓存命中，把这些名称升级成授权或证据已经存在。名称用于定位待核对对象，不承担认证、授权或最终判断。

文件内结论与外部前置条件怎样报告

检查层当前字节可以直接证明什么必须单独报告什么
容器接受前导、目录、范围、资源上限和确定性载荷满足 END-FMT不解释六个语义记录是否构成可接受内容
Profile 接受记录种类、字段、枚举、排序和引用满足 END-P1END-P1 单文件最高只能声称 Profile 接受
内容接受文件内结构提供待核对候选与选择器外部规则、授权、身份、签名和 Iknem 是否取得且精确绑定
满足判断目标、判据和要求可以被读取现实观察、有效证据、覆盖度和结果计算
最终决定可以定位具名决定主体有效决定权、输入结果、范围和 accepted / rejected / deferred

外部前置条件缺失、未执行、版本不匹配、错绑、撤销或无法核对时，报告必须明确写成“未满足”或“未求值”。不得省略该层，也不得把调用者断言改写为通过。

四项条款怎样关闭原有矛盾

条款开发者必须遵守失败后果
END-CON-006分开报告文件内结论与外部前置条件，并声明单文件最高可声称层次缺失或未求值层次不能隐式通过
END-AUT-002权威、规则和证据名称都只作为待核对选择器缺失、错绑、拒绝、延后、过期或撤销不能产生已授权投影
END-ID-002授权、签名、Iknem、满足与最终决定不进入 Endem 内容身份字节改变必须形成新身份；授权变化只产生新伴随评估
END-FMT-015END-P1 不内联政策、目录、决定、签名、证据或验证材料读取器不能从环境或网络搜索补齐文件含义

两个字节完全相同、外部授权证据不同的 Endem 必须进入同一内容身份等价组。相反，只要六个语义面或规范字节改变，就必须建立新身份，不能把语义修订伪装成授权状态更新。

外部伴随机制提供哪些工程参照

外部机制支持的分层纪律Noemion 不直接采用什么

ELF SHF_OS_NONCONFORMING未知且需要特殊处理的关键语义必须拒绝，不能静默忽略不把 ELF 节、标志或装载语义移植到 Endem

这些资料共同说明对象、描述对象的陈述、验证材料、本地信任政策和依赖方决定可以保持不同生命周期。它们只提供反例与分层参照，不定义 Endem 的伴随物理格式。

当前还不能承诺哪种伴随 Profile

SCN-028 至 SCN-030、语义向量的外部前置条件矩阵，以及缺失和错绑授权诊断已经验证规范关系。它们不能证明伴随资料怎样封装、分发、撤销、缓存或跨系统查询。

待定内容：授权伴随关系、权威目录、政策语言、规则登记表、决定事件、撤销分发和符合性报告仍没有物理 Profile。当前改动只修正规范边界，不创建新文件、包络、服务或组件。

Endem 规范指南从六个语义面、判据和开发者问题继续核对内容要求。 ADR-0023 · 文件能读，不等于内容成立继续区分容器、Profile、内容、满足和最终决定。 ADR-0029 · 身份已确认，不等于操作已获准继续核对外部授权绑定需要的主体、范围、时效和结果。 ADR-0036 · 移除原文，就得到另一份制品继续核对形成版、发布版、新身份和来源伴随关系。

术语与读音：“内容身份”“授权伴随关系”“外部前置条件”“截止点评估”分别指对象、关系、依赖与时间判断，职责和口头表达可区分；当前决定不引入新的发行专名。
