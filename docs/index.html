文档边界

本页进入六个独立 HTML 文档。文档负责解释、阅读路径和当前状态；正式定义仍链接到各权威规范页面。

成熟度与引用纪律

研究、论文、专利、软著和标准草案引用项目内容时，必须记录页面路由、版本或提交标识、内容状态和对应权威来源。项目愿景不能作为实验结论，候选类型名不能作为冻结 ABI，页面中的未来流程也不能作为已经发布的软件能力。

后续文档将逐步为关键主张增加规范条款、ADR、测试、基准、数据来源和失败案例链接，以形成可审计的证据链。
