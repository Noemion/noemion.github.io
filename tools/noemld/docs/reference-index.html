类别当前解释权威化条件

追溯要求

未来每个稳定术语条目应链接到唯一规范定义、相关 ADR、实现接口、诊断类别和测试向量。别名必须标明首选名称和废止状态；当前开发阶段不保留旧路由或兼容语义。若页面与规范冲突，以规范成熟度和经批准决策为准，并修正页面。
