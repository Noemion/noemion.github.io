工具链概要设计

这些工具按对象生命周期组织。认证工具执行规范测试；对象工具检查和变换文件；文本工具提供可阅读表示；编译与链接工具形成确定性对象；发布工具冻结覆盖、依赖和签名闭包；运行工具分别记录装载、Harness、Runtime 和评价证据；模型工具只产生不可信候选。

层次设计责任不得跨越的边界
合规与验证把规范条款转成可执行测试、诊断和一致性证据不能用单一实现行为反向定义规范
对象与文本表示安全读取、转换、比较、汇编和反汇编对象不能绕过结构、类型、权限和完整性检查
编译与链接确定性生成、解析符号、应用重定位和形成闭包不能调用模型解决规范要求确定处理的冲突
发布与运行裁剪、覆盖审计、签名打包、可信装载和追踪不能静默扩大权限、依赖或披露范围
模型工程数据、训练、评估和部署候选模型能力模型输出始终不可信，不能直接决定对象布局

这些职责是否需要独立工具

23 个名称是可审计的职责入口，不是必须各自维护一套进程、仓库或重复解析器。实现阶段应先共享结构定义、安全读取器和报告结构，再根据权限、失败原子性、使用频率与发布边界决定独立命令、子命令或库接口；不能因为目录已有名称就证明独立工具有价值。

职责组保留价值适合的最小形态合并或停止条件
验证、链接、覆盖、打包、执行分别承担安全拒绝、依赖闭包、证据闭包、签名封装和会话控制，权限与失败责任不同。独立命令面或服务边界，共享底层对象库。可以同仓实现，但不能合并掉验证结论、签名身份、会话权限或 Acceptance Decision 边界。
查看、预算、符号、比较、观察、认证为开发者、持续集成和审计提供只读视图，避免只能通过变换工具理解对象。独立子命令或统一只读工具集；机器输出结构保持稳定。若独立命令只复制同一报告且没有不同受众、性能或权限需求，应合并为子命令。
格式化、汇编、反汇编、编译、分析、归档、裁剪与受控变换输入语言、是否改写数据、事务语义和下游产物不同，分名有助于明确失败后是否可以提交输出。共享 Parser、Writer 与变换库的多个命令面。noemtransform 若不能证明通用事务框架优于明确的专用操作，应缩减或并入具体工具；其他薄封装同样处理。
数据、训练、评估与量化保存 Noemion 特有的来源血缘、泄漏检查、资格记录与发布身份链。先作为现有模型工程系统的可复现适配层和清单规范。若没有超出现有工具的项目特有约束或可复现证据，不建设重复训练平台；Horizon Engine 不能证明增益时停止投入。

规范合规

noemcertify执行规范条款、基准样例、畸形样例和跨工具一致性测试。

安全对象工具

noeminspect安全查看 Header、Section、Symbol、Relocation、NIR 与披露结构。 noemvalidate分层验证结构、语义、覆盖、策略和信任链。 noemtransform受约束地复制、抽取、删除和重建对象内容。 noembudget分析 Section、Segment、闭包和披露预算占用。

文本 IR 工具

noemassemble将规范文本 NIR 确定性汇编为 NOBJ。 noemdecode将对象反汇编为可再次汇编的规范文本。 noemformat统一 NSL 与文本 NIR 的稳定表现形式。 noemcompare区分布局、调试、信任元数据和运行语义差异。

编译与链接

noemcompile驱动 Deterministic、Model-assisted 与 External-generation 前端及确定性 Noesis Core。 noemanalyze检查 NSL、NIR、覆盖和策略问题，不生成对象。 noemarchive建立确定性 NOBJ 归档和符号索引。 noemsymbols检查符号定义、引用、版本和可见性。 noemlink解析符号、应用重定位，并输出 Linked Object、可重定位 NOBJ 或 HOBJ。

发布与运行时

noemreduce分离开发调试信息并证明发布语义不变。 noemcoverage分别建立 Release Coverage Proof 与 Run Evidence Closure。 noembundle冻结发布闭包，生成签名请求并核对外部签名回填。 noemexecute验证签名包，驱动 Harness / Runtime，并在证据闭合后形成 Acceptance Decision。 noemobserve规范化运行事件并报告 Trace 完整性与证据强度。

模型工程

noemdataset构造有许可、有血缘、无泄漏的数据集快照。 noemtrain编排 Horizon Engine 适配、多任务训练和教师蒸馏。 noemevaluate离线形成模型资格记录与最终 Agent 场景评估。 noemquantize导出、量化、端侧基准并产生未签名 Model Package Candidate。
